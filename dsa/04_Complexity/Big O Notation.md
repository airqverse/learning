# Big O Notation

Big O notation is used to describe the upper bound of an algorithm's running time or space requirements in terms of the input size $n$.

## Key Rules
1. **Drop Constants**: $O(2n) \rightarrow O(n)$
2. **Drop Less Significant Terms**: $O(n^2 + n) \rightarrow O(n^2)$
